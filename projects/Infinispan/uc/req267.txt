[ISPN-4919] Configuration templates <p>Currently there is a 1:1 relationship between configuration and named caches. While the programmatic API does have the ability to .read() an existing configuration to create a new one, the declarative config does not.<br/>
We should introduce the concept of configuration inheritance, e.g.:</p>
